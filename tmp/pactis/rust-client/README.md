# Pactis Rust Client (Draft)

High-performance CLI and library for packaging and publishing Pactis UI components.

Goals
- Fast file discovery and packaging for large components
- Robust publishing with retries and resumable uploads (future)
- Optional incremental diffs to reduce network and storage
- Clean typed API, embeddable in other tools

Status
- This is a planning stub. See architecture and CLI specs below.
