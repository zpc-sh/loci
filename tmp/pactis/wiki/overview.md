# Overview

The Pactis JSON-LD paradigm uses linked data to represent specifications, components, and tests as portable, machine-readable artifacts. This enables:

- Semantic interoperability: contexts define unambiguous meaning.
- Tooling integration: export, validate, execute, and package with standard commands.
- Traceability: test results are persisted as JSON‑LD alongside specs.
