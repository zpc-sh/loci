# Pactis JSON-LD: High-Level Overview

Pactis uses JSON-LD to encode specifications, components, and test contracts as semantically-rich, portable artifacts. This unlocks:

- A shared vocabulary for resources, blueprints, and UI components
- Automated validation and execution of test suites
- Packaging and distribution of a reusable specs library

At a glance:
- Define specs with clearly scoped contexts (data + meaning)
- Validate or convert from Markdown to JSON-LD
- (Optionally) execute test suites and store results as JSON-LD
- Bundle contexts, templates, examples, and tests for teams or partners
