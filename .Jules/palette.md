## 2024-05-18 - @clack/prompts Cancellation UX\n**Learning:** When users hit `Ctrl+C` in multi-prompt flows, subsequent prompts are skipped but the script might continue running if `clack.isCancel()` is not explicitly checked for each prompt. Furthermore, a strict typing issue requires a string message passed to `clack.cancel()`.\n**Action:** Always verify cancellation via `clack.isCancel(value)`, call `clack.cancel("Cancelled.")` with a message to give feedback to the user and prevent typing errors, and exit the process using `process.exit(0)` to completely stop execution.
